# fedorafy
Post installation script for fedora

#### Start the post-installation setup by running the following command

`wget https://raw.githubusercontent.com/Spicy-Steve/fedorafy/main/fedorafy.sh && sudo sh fedorafy.sh`

<br>

## For unattended installs, use one of the following

### AMD

`wget https://raw.githubusercontent.com/Spicy-Steve/fedorafy/main/unattended/amd/fedorafy-unattended-amd.sh && sudo sh fedorafy.sh`

##### GAMING ADDITIONS

`wget https://raw.githubusercontent.com/Spicy-Steve/fedorafy/main/unattended/amd/fedorafy-unattended-amd-gaming.sh && sudo sh fedorafy.sh`
